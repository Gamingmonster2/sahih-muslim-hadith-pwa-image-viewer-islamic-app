# 📚 Sahih Muslim PWA Hadith Viewer | تطبيق صحيح مسلم الرقمي

### [ EN ] Project Overview
A high-performance **Progressive Web App (PWA)** built to provide a seamless **image-based browsing** experience for **Sahih Muslim** (The authentic collection of Hadith). This project is optimized for **SEO**, **Offline Access**, and **Fast Performance** using modern web technologies.

**🔍 Keywords:** Sahih Muslim, Hadith, Islamic Web App, PWA, Image Viewer, Muslim bin al-Hajjaj, Sunnah, Arabic Books.

---

### [ AR ] نظرة عامة على المشروع
تطبيق ويب تقدمي (PWA) عالي الأداء متخصص في **عرض أحاديث صحيح مسلم** بنظام الصور. تم تصميم هذا المشروع ليكون **مرجعاً رقمياً** سريعاً يدعم **التصفح بدون إنترنت** (Offline) مع التركيز على **تحسين محركات البحث (SEO)** لضمان وصول المحتوى عالمياً.

**🔍 كلمات مفتاحية:** صحيح مسلم، أحاديث نبوية، تطبيق ويب إسلامي، أرشيف رقمي، السنة النبوية، مسلم بن الحجاج.

---

### 🚀 Key Features | المميزات الرئيسية
* **PWA Ready:** Installable on Android, iOS, and Desktop.
* **Offline Mode:** Browse pages without an internet connection.
* **Image Optimization:** Lazy loading for ultra-fast flipping.
* **Bilingual Interface:** Support for Arabic (RTL) and English (LTR).
* **Dark Mode:** Professional UI with Material Design principles.
